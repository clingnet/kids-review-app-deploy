# 期末复习助手 — 需求文档 (PRD)

> 版本:v1.0 · 2026-06-20 · 每次迭代需同步更新本文件
> 一句话:家长拍试卷照片,AI 自动分析知识点与薄弱点,孩子在科幻界面里针对性学习、自我练习、与 AI 老师问答、做模拟卷。

## 1. 背景与目标
- **用户**:程令(家长)+ 五年级孩子(单用户,自托管,无需账号)。
- **科目**:数学 / 语文 / 英语。**场景**:期末复习。
- **设备**:iPad Pro,Safari PWA(加到主屏全屏)。
- **目标**:把"拍照→分析→针对性学练→查漏补缺→模拟考"做成闭环,越用越懂孩子。
- **约束**:可整体迁移(Docker+挂载卷,海外→国内→家用主机);隐私自托管;家长每周投入少。

## 2. 信息架构(三大模块)
顶层导航:**首页 | 知识图谱 | AI老师**

### 2.1 首页「我的学习舱」(以孩子为中心)
> 不放上传、不放模拟考。聚焦"我学得怎样 + 针对性学/练"。
- **F1 学习概览 HUD**:综合掌握度、各学科掌握度、连续学习天数/成就、近 7 天掌握度趋势、薄弱点数量。
- **F2 今日推荐学习**:基于薄弱点/掌握度推荐"今天练什么"。
- **F3 薄弱点速攻**:薄弱点列表,每个可「讲解+练习」或「问 AI 老师」。
- **F4 自我练习(随练)**:选范围(学科/知识点/薄弱点/错题本)→ 即做即判 → 解析 → 错题入本。**不计入正式考试**。
- **F5 最近动态**:最近练习、错题回顾入口。

### 2.2 知识图谱(全局知识中心)
- **F6 试卷/错题上传**:拍照/相册多选 → 上传 → **异步自动分析**(分析时全局提示"分析中,知识图谱/薄弱点稍后更新")。
- **F7 知识图谱可视化**:知识点关联图,按掌握度发光;科目/掌握度筛选。
- **F8 节点详情**(点击知识点):📖知识点详解 / 📝出过的题目 / ❌做错过的题目(你的答案 vs 正确答案)/ 🤖找 AI 老师讨论。
- **F9 全局分析报告**:重复度/难度/超纲度概览 + 自动总结的薄弱点。
- **F10 异步更新提示**:知识图谱/薄弱点页默认显示上次结果;若有新上传分析完成——正在用该页则提示"是否更新",不在用则静默更新。

### 2.3 AI老师(问答 + 命题)
- **F11 问答辅导**:多轮对话(文字/语音/拍照),一对一讲解;可带某知识点上下文进入(从首页/图谱跳入);多轮讲懂 → 出题考察 → 判定掌握 → 回写掌握度。
- **F12 出模拟卷**:**三来源**(均支持手动/自动):① 知识图谱选考点 ② 错题选考点 ③ AI 对孩子整体分析(全自动)。配题量/题型/难度/总分 → 生成卷子 + 答案解析。
- **F13 卷子结果**:查看 / 显隐答案 / 打印 PDF。每卷带**唯一编号**(SJ-XXXXX)印在卷上。
- **F14 拍照判分(闭环)**:孩子做完打印的卷子 → 家长拍照上传 → AI 识别编号自动对应到哪张卷 → 逐题判分 → 成绩存档 + 错题自动入错题本。
- **F15 每日一练**:首页入口,自动挑薄弱知识点随机出题、随时可中断。
- **F16 性能**:讲解/详情缓存、出题/组卷优先题库(秒出),"刷新"才调 AI;上传可最小化到后台。
- **F17 公网访问**:部署 443/HTTPS(自签证书 + 访问密码),免 SSH 直接打开。

## 3. 关键用户故事
- 作为家长,我拍一沓试卷上传(在"知识图谱"页),系统自动分析,我不必等着——分析在后台跑,页面提示"分析中"。
- 作为孩子,我打开首页就看到自己各科掌握情况和今天该练什么,点"自我练习"做几道题立刻知道对错。
- 作为孩子,我在知识图谱点开"通分",看到它的讲解、我以前做过/做错的题,还能让 AI 老师给我讲。
- 作为家长,我让 AI 老师基于孩子薄弱点出一套模拟卷,打印给孩子做。

## 4. 非功能需求
- **可迁移**:Docker Compose + 数据全在挂载卷;拷目录即迁移。
- **隐私**:数据仅在自托管机器 ↔ 模型 API(OpenRouter)之间。
- **成本**:大小模型分层 + 图片压缩 + 缓存;一轮分析几分钱级。
- **性能**:上传异步;前端图片压缩;模型分析容忍数十秒,前端有进度/提示。
- **手写识别**:重点保障(用 Qwen-VL,可切 Gemini 兜底)。

## 5. 里程碑
- **M1 后端核心**(已建+真机验证):上传/提取/分析/薄弱点/讲解/模拟卷/辅导。
- **M2 三模块落地**(本期):异步分析、首页仪表盘、节点详情、自我练习+错题本、AI 问答 chat、前端接真实后端、部署上线。
- **M3 迭代**:语音 ASR/TTS、在线答题卡判分、激励体系、知识图谱图形增强。

## 6. 范围外(暂不做)
- 多用户/账号体系;App Store 原生上架;语音实时双工(先按住说话/暂不接)。
